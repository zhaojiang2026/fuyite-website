# 福伊特企业管理官网

## 项目概览

**企业名称**: 福伊特企业管理有限公司 (FUYITE)
**定位**: 人力资源服务 + 劳务派遣专业机构
**价值观**: 专注、高效、共赢
**愿景**: 成为长三角具有影响力的专业人力资源供应商

## 技术栈

- **Framework**: Next.js 16 (App Router)
- **Core**: React 19
- **Language**: TypeScript 5
- **UI 组件**: shadcn/ui (基于 Radix UI)
- **Styling**: Tailwind CSS 4

## 页面结构

| 路径 | 页面 | 说明 |
|------|------|------|
| `/` | 首页 | 品牌展示、核心业务、优势介绍 |
| `/about` | 关于我们 | 企业介绍、文化、团队实力 |
| `/services` | 服务项目 | 5大核心业务详解 |
| `/contact` | 联系我们 | 联系方式、在线咨询表单 |

## 核心服务

1. **人力资源管理咨询** - 战略规划、体系诊断、劳动关系咨询
2. **人力资源业务外包** - 薪酬外包、RPO、岗位外包
3. **人才招聘和派遣** - 猎头、批量招聘、劳务派遣
4. **人力资源培训及开发** - 定制培训、HR能力指导
5. **个人职业及发展规划** - 职业咨询、社保公积金代理

## 部署

本项目已配置 Vercel 部署，连接 GitHub 后自动部署。

### 自定义域名配置

1. 在 Vercel 项目 Settings → Domains 中添加你的域名
2. 在阿里云 DNS 控制台添加 CNAME 记录指向 Vercel 提供的域名
3. 等待 DNS 生效（通常 5-30 分钟）
